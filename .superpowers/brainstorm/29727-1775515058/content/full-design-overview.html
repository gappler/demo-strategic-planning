3. Interactive Features

🔀
Scenario Switcher
Click sidebar buttons to switch. Cards animate — shared objectives cross-fade with updated KR targets, scenario-only cards slide in/out. Sidebar context and resources update instantly.
🏷️
Function Filter
Color-coded pill bar filters cards by Marketing, Product, Operations, Finance. Multiple filters can be active. Resource summary adjusts to show only visible functions.
📋
Card Expand
Click any card to expand inline. Shows full KR detail with from→to metrics, timeline, team owner, and the strategic challenge description from the brand definition.
⚡
Objective Toggle
Each card has an on/off toggle. Disabling grays the card out and removes its resources from the sidebar totals. Lets you model "what if we skip this objective?" in real time.

4. Technical Approach

1
Single HTML File
No dependencies, no build. All CSS + JS embedded. Opens in any browser.
2
Data-Driven
All OKR content in a JS object at top of file. Easy to edit scenarios, objectives, and KRs.
3
CSS Transitions
Smooth card animations on scenario switch. Fade, slide, and color transitions for visual clarity.
