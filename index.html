A suite of five interactive tools built to help Verdana's leadership navigate the growth plateau and chart a path forward. Each tool was built with a single prompt.

5 tools
1 OKR Goal Setting Tool Built

Verdana needs to align 60 people around annual objectives, but the goals change depending on which strategic scenario plays out. This tool lets leadership switch between Base ($9.5M), Upside ($12M), and Downside ($7.5M) scenarios and see how objectives, key results, and resource requirements cascade across Marketing, Product, Operations, and Finance.

The Prompt
"Build an OKR and annual goal setting tool for Verdana. Organize objectives by function across three strategic scenarios. Let me toggle objectives on/off and see resource impact in real time."

3 Scenario Modeler Built

Verdana can't pursue wholesale, new products, and product rationalization all at once. This modeler lets leadership toggle three strategic moves on/off, tune assumptions with sliders, and see 3-year quarterly projections for revenue, margins, investment, headcount, and risk. Pin named scenarios and compare them side by side.

The Prompt
"Build a scenario modeler for Verdana. Model four strategic paths: stay the course (DTC only), expand into wholesale, launch two new products, or cut the underperforming SKU and reallocate resources. For each scenario, project 3-year revenue, margins, required investment, team headcount, and risk factors. Let me compare scenarios side by side and blend them — what if we do wholesale AND cut a SKU? Output as an interactive HTML file."

How This Was Built

Methodology

This toolkit was built by following a deliberate sequence:

  1. Business definition first. Before any tool was prompted, we defined Verdana — its revenue, margins, team size, product lines, competitive landscape, and the specific growth plateau it faces. Claude Code can’t build something useful without knowing what “useful” means for your business.
  2. Real-world data, not placeholders. Every number in these tools — the $7.2M revenue baseline, the 62% gross margins, the 9 competitor brands — comes from a structured data layer built before the first tool was touched. The tools read from this data; they don’t invent it.
  3. Sequential, compounding tools. Each tool feeds the next. The OKR tool established the goals. The market sizing calculator quantified the levers. The scenario modeler combined those levers into strategic paths. The competitive map provided external context. The executive summary synthesized all four into a board-ready recommendation. Remove any one layer and the final output loses fidelity.
  4. One prompt per tool, built with Claude Code. Each of the five tools was generated from a single, well-crafted prompt — but “well-crafted” is the operative word. The prompt quality depends entirely on the groundwork laid in steps 1–3.
